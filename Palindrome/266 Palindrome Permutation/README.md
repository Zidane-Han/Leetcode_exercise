# Palindrome Permutation (Easy)

## Description

Given a string, determine if a permutation of the string could form a palindrome.

**Example 1:**

```c++
Input: "code"
Output: false
```

**Example 2:**

```c++
Input: "aab"
Output: true
```

**Example 3:**

```c++
Input: "carerac"
Output: true
```
